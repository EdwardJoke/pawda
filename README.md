# pawda
![Build](https://github.com/EdwardJoke/pawda/actions/workflows/rust.yml/badge.svg)

## Description
Pawda is your next-gen command to replace your `pwd` commands.

## Overview

### Why Pawda?
- Pawda is designed to be a simple, fast, and efficient command to replace your `pwd` commands.
- Pawda is written in Rust, which means it is fast, secure, and cross-platform.

### Installation
- Pawda is available on [GitHub](https://github.com/edwardjoke/pawda).
- You can install Pawda by:
1. Cloning the repository: `git clone https://github.com/edwardjoke/pawda.git`
2. Navigating to the project directory: `cd pawda`
3. Installing Pawda: `cargo install -path .`

## Usage
- You can use Pawda by simply typing `pawda` in your terminal.
